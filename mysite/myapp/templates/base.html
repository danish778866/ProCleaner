This tool allows you to clean files containing strings.
Assumption: A line in the file is treated as one string.

Please follow the below steps:
  1. Choose a file and click Submit.
    • No files uploaded yet? Choose a file to upload, and click Submit.
    • Already uploaded files earlier? Select a file from the list, and click Submit.
  2. Pick cleaning action, and click Execute.
  3. Check the correct label(s) out of the candidate match pairs, and click Clean Up.
  4. Download the cleaned file, and the rules used for the clean operation.
  5. On the Navigation bar, click Upload to go back to start page, or Download to see a list of all the files that you can download.
