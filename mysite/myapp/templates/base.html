Modal Header

This tool allows you to clean files containing strings.
Assumption: A line in the file is treated as one string.

Please follow the below steps:
  1. Choose a file and click Submit.
    • No files uploaded yet? Choose a file to upload, and click Submit.
    • Already uploaded files earlier? Select a file from the list, and click Submit.
  2. Pick action, and click Execute.
  3. If chose to clean strings:
    • Check the correct label(s) out of the candidate match pairs, and click Clean Up.
    • Download the cleaned file, and the rules used for the clean operation.
  4. If chose to profile:
    • Profiling Statistics will be shown regarding string lengths and the words containing them.
  5. If chose to find errors:
    • Different kinds of possible errors alongwith their pervasiveness will be shown.
  6. On the Navigation bar, click Upload to go back to start page, or Download to see a list of all the files that you can download.
